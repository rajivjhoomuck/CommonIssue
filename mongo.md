

# Mongo DB cheatsheet

Montrer une collection :
*show collections*

Utiliser une collection / Créer une collection :
*use macollection*
Si elle existe pas il la crée

Clear une collection

Insert : 
*db.acteurs.insert({nom:"Johansson", prenom:"Scarlett"})*

Simple query :
**db.acteurs.find()** 

Delete :
**db.produits.remove({mill: 600})**
